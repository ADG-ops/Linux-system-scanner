# 🐧 Linux System Scanner

A lightweight Python utility designed to monitor and report key system metrics. This project demonstrates system administration automation using Python.

---

## Features
- **CPU Monitoring:** Real-time load percentage.
- **Memory Tracking:** Total and used RAM (in GB).
- **Disk Usage:** Storage analysis for the root partition.
- **System Info:** OS version and hostname identification.
